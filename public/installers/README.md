# Instaladores del conector POS

Publicar en esta carpeta los instaladores finales con estos nombres:

- `paseo-ticket-printer-mac.dmg`
- `paseo-ticket-printer-windows.zip`
- `paseo-ticket-printer-windows.exe` cuando se genere el instalador ejecutable final

La pantalla `Configuración > Impresora POS` descarga directamente esos archivos.
